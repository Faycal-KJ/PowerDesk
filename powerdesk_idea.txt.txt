UI Style

I'd avoid copying Windows 11 exactly. Instead, aim for something between VS Code, Arc Browser, and Notion:

Minimal chrome (less wasted space)
Soft rounded corners
Dockable side panels
Command Palette (Ctrl+P or Ctrl+Shift+P) for nearly every action
Keyboard-first navigation for power users
Light and dark themes from day one
Smooth but subtle animations

One addition I'd make that could set PowerDesk apart

Instead of thinking of it as a "better Explorer," think of it as the VS Code of file management.

VS Code didn't win because it had more features than every editor on day one—it won because it had:

A clean, extensible core.
A great user experience.
A powerful extension ecosystem.

If PowerDesk follows the same philosophy—a polished core with plugins, automation, and eventually AETHER integration—you could grow it from a useful utility into the centerpiece of a broader productivity ecosystem rather than just another file manager.
