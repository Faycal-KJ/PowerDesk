# Contributing to PowerDesk

PowerDesk is a personal side project maintained by one developer (me) in their free time.

## What's Accepted

- **Bug reports** — please open an issue with steps to reproduce
- **Feature suggestions** — open an issue describing what you'd like

## What's not Accepted for now

- Large pull requests or new features
- Refactoring suggestions
- Adding new dependencies

I may fix bugs and review issues when I have time, but there is no guarantee that requests or pull requests will be accepted.

## Building from source

```bash
git clone https://github.com/PowerDeskApp/PowerDesk.git
cd PowerDesk
npm install
npm run electron:dev
```
