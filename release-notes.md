## PowerDesk v0.1.0

First public release of PowerDesk — a modern file manager for Windows.

### Features

- Tabbed file browser with dual-pane mode
- Grid, List, and Gallery view modes
- Smart search with natural language filters
- Transfer Center with pause/resume/cancel
- Universal undo/redo
- File preview for 40+ file types
- File Inspector with EXIF metadata
- Folder Analysis with duplicate detection
- Clipboard Manager
- Built-in tools (QR generator, terminal, color picker)
- Tags and color labels
- Workspace profiles
- Multi-window sync
- 8 color themes with full UI customization
- Plugin system with 15 extension points

### Installation

**Portable:** Download `PowerDesk_Portable.zip`, extract, run `PowerDesk.exe`.
**Installer:** Download `PowerDeskSetup.exe`, run to install.
